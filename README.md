## Setting up for deployment

### Prequisites
- Node JS
- React JS
- Firebase

> npm install

For Development:

> npm start

For Production Build:

> npm run build

### Create Firebase App

- Change Firebase Settings in /src/firebase.js

- deploy on firebase

 > firebase deploy
